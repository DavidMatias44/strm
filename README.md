# strm (string modifier)

A command-line tool for modifying strings; removing vowels, removing duplicate characters and sorting the result string.

I needed to remove vowels and duplicate character from strings, so I created this. It helped me review the `argc` and `argv` parameters functionality. Also, I recently discovered the stb header style, so I decided to use it.
 
I might add some new features and improve the code, but I do not know I just did it because I was bored.

## Quick start

Compile and run:

```shell
./build
./main <string>
```

To see available options:

```shell
./main --help
```
